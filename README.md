# 멀티모듈로 주문 도메인 만들기
최근 사내에서 개발 연구중인 과제가 멀티모듈로 구조를 잡으면 괜찮겠다 생각했습니다. 
그래서 주문 도메인 생성하고 개발하는것을 통해 멀티모듈을 익히고자 합니다.

## 추가 연구 및 적용 사항
 * 사가패턴 적용
 * ELK 모니터링 적용
 * 도커 배포 적용
 * 카프카
 * 도메인 이벤트


### 모듈 설명
멀티모듈 구조 설계는 https://techblog.woowahan.com/2637/
다음을 참고하였습니다.

* core-api : 순수 Java의 기능을 사용한 모듈 주로 공통 모듈 type 이나 Util 을 정의.
* domain : 도메인의 순수 비지니스로직과 인프라스트럭쳐를 정의합니다.
* internal-api : 외부로부터 요청을 받을 수 있게 API 를 정의합니다.
